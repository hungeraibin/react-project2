# 基于React的在线Todo应用

> A React.js project

- 功能介绍：这是一款操作简便TodoList应用。支持多用户在线注册、登录账号、找回密码功能，数据存储同步到云，方便随时随地添加或删除自己的待办项目。
- 技术细节：使用create-react-app初始化项目，使用ES6语法。组件上划分上，遵循单一原则封装更多的组件。使用LeanCloud实现表单的登录、注册功能，存储用户数据。
- 技术栈：React，ES6，LeanCloud

## build

```
create-react-app . 

npm start

npm run build
```
